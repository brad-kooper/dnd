# DnD React App

Simple React starter app.

# Technologies

* ECMAScript 2015 (aka ES6) support through Babel
* React 16.x
* Webpack 3.x
* CSS and Sass support
* Image loading support
* React router support
* Redux support
* ECMAScript 2016 (aka ES7) support

# Getting Started

1. `yarn install`
2. `yarn start`
3. view in browser: `http://localhost:8080/`
